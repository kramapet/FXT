FXT
===

To install development version:
```shell
git clone https://github.com/kramapet/FXT 
cd FXT/
git checkout 0.2.dev
pip install -e .
```
or directly:
```shell
pip install git+https://github.com/kramapet/FXT@0.2.dev
```

Run by (use config.json from current working directory):
```shell
fxt
```
